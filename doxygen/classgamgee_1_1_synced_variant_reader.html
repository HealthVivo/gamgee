Detailed Description

template<class ITERATOR>
class gamgee::SyncedVariantReader< ITERATOR >

Utility class to read multiple VCF/BCF files with an appropriate iterator in a for-each loop.

This class is designed to parse the files in for-each loops with the following signature:

for (auto& record : SyncedVariantReader<SyncedVariantIterator>{filenames})
do_something_with_record(record);

You can also use it with stdin or any other stream by using the default constructor or passing in an empty string for a filename, like so:

for (auto& record : SyncedVariantReader<SyncedVariantIterator>{filename1, stream2})
do_something_with_record(record);
