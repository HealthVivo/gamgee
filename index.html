Welcome to Gamgee home

Gamgee is a C++ interface for HTSLIB (samtools/htslib). It provides an easier, memory-safe and object-oriented way to access the high performance HTSLIB features. Gamgee is designed to keep the original performance of HTSLIB while providing nicer and safer interfaces.

Build prerequisites

In order to build Gamgee you need Boost.Build V2 installed in your system. Follow the installation steps below so you can use b2 to build Gamgee.

Installing Boost.Build V2 from source

Run the following commands to checkout and install Boost.Build V2:

git clone https://github.com/boostorg/build.git
cd build
./bootstrap.sh
sudo ./b2 install

Download and installation

Once installed Boost.Build V2, run the following commands to install Gamgee: 

$ git clone https://github.com/mauriciocarneiro/gamgee.git
$ cd gamgee
$ b2 gamgee
